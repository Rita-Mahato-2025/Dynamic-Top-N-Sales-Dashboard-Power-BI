# 📊 Dynamic Top N Sales Dashboard – Power BI

📌 Project Overview

Built an interactive Power BI dashboard that dynamically displays Top N selling products based on user selection using What-If parameters.

🎯 Business Problem

Business users frequently requested different Top N views (Top 3, Top 5, Top 10). Instead of building multiple reports, a dynamic solution was designed.

🛠 Tools Used

 - Power BI

 - DAX

 - What-If Parameters
 
  - Superstore Dataset (Kaggle)

📈 Key Features

 - Dynamic Top N selection

 - Product ranking using RANKX

 - Interactive slicers

 - Clean, business-friendly dashboard

💡 Business Impact

Reduced report duplication and improved decision-making flexibility for stakeholders.
